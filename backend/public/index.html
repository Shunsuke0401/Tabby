Try it yourself

⬇️ Download the Tabby extension
  1. Unzip the download.
  2. Go to chrome://extensions and turn on Developer mode (top-right).
  3. Click Load unpacked and pick the unzipped extension folder. Open Tabby from the toolbar.

It works immediately — Tabby talks to this deployed backend, so there's no API key or setup on your end.
